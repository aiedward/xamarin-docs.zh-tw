# <a name="certified-consulting-partners-to-deliver-mobile-solutions"></a>提供行動解決方案的認證諮詢合作夥伴

> [!TIP]
> **從2018年6月30日起，Xamarin 諮詢合作夥伴方案已與 Microsoft 合作夥伴網路合併。**
>
> 開始使用 [Microsoft 合作夥伴網路](https://partner.microsoft.com/)：
>
> * 如果您還沒有註冊 Microsoft 合作夥伴網路成員，請 [註冊](https://partner.microsoft.com/membership) 成為合作夥伴。  
> * 藉由完成 [應用程式開發專長認證](https://partner.microsoft.com/membership/application-development-competency) 及/或 [雲端平臺專長認證](https://partner.microsoft.com/membership/cloud-platform-competency)，來示範您的專業知識。
> * 更新您個人資料的專業標籤，以納入 Xamarin、Android、iOS、Mobile Apps 和行動裝置應用程式開發，讓您可以被視為 [Microsoft 解決方案提供者](https://www.microsoft.com/solution-providers/home)。
