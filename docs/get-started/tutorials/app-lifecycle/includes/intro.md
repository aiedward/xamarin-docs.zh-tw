- [建置您的第一個 Xamarin.Forms 應用程式](~/get-started/first-app/index.md)快速入門。
- [StackLayout](~/get-started/tutorials/stacklayout/index.yml) 教學課程。
- [項目](~/get-started/tutorials/entry/index.yml)教學課程。
