## <a name="xamarintvos-application-fundamentals"></a>TvOS 應用程式基本概念

本節中的檔可讓您瞭解在建立 tvOS 應用程式時，將會使用之 tvOS 的基本概念和功能。

強烈建議您先完成[Hello，tvOS 快速入門手冊](~/ios/tvos/get-started/hello-tvos.md)，因為它涵蓋了我們將在每篇文章中使用的重要概念和技巧。
