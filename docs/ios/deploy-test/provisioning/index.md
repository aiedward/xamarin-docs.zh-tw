成功安裝 Xamarin.iOS 之後，iOS 開發作業的下一步就是佈建您的 iOS 裝置。 本指南將探索如何要求開發憑證和設定檔、使用應用程式服務，以及將應用程式部署至裝置。

權利是特殊的應用程式功能和安全性權限，其授與對象是已正確設定來使用這些功能和權限的應用程式。 本指南將說明如何使用它們來佈建您的 iOS 應用程式。
