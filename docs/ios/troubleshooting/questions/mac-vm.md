# <a name="can-i-use-a-mac-vm-with-xamarin"></a>我可以使用 Mac VM 搭配 Xamarin 嗎？ 

## <a name="mac-hardware"></a>Mac 硬體
肯定請小心遵循 Apple 的使用條款。

## <a name="non-mac-hardware"></a>非 Mac 硬體
不要Apple 的使用規定不允許這種使用方式。
