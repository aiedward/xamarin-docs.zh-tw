# <a name="get-started-with-macos-mojave"></a>開始使用 macOS Mojave

本檔說明如何使用 Xamarin 建立 macOS Mojave 應用程式。 它會討論如何下載 Xcode 10 和更新 Visual Studio for Mac。

## <a name="download-and-install"></a>下載並安裝

1. **安裝最新的 Xcode 10 Beta** –註冊的 apple 開發人員可以從 [Apple 開發人員入口網站](https://developer.apple.com/download/)下載並安裝最新版本的 Xcode 10。

2. **執行 Xcode 10** –執行 Xcode 10，然後更新和執行 Visual Studio for Mac;它會安裝一些 Xamarin 所需的工具。

3. **更新 Visual Studio for Mac** -使用 Visual Studio for Mac 的最新穩定版本，以及 [Xamarin. Mac 5.0](https://github.com/xamarin/release-notes-archive/blob/master/release-notes/mac/xamarin.mac_5/xamarin.mac_5.0.md) 或更新版本。

4. _ (選擇性) _ **在您的 Mac 上安裝 macOS Mojave** –

   > [!TIP]
   > 即使您的應用程式不會使用任何新的 macOS Mojave Api，但請務必使用 macOS Mojave SDK 來建立它，並測試它以確定一切都能如預期般運作。 如果應用程式不會呼叫任何新的 Api，您可以使用 macOS Mojave SDK 重新編譯它，並在不升級 Mac 作業系統的情況下進行測試。
   >
   > 將 Mac 升級為 macOS Mojave 之前，請先建立並測試呼叫新 macOS Mojave Api 的 Xamarin. Mac 應用程式：
   >
   > - 閱讀 [Apple 的](https://developer.apple.com/download/) 作業系統更新版本資訊。
